## What is this?

`issuepack` is a multi-agent system that reads your GitHub issues, touches your code, and confidently opens a PR.

Accuracy not guaranteed. Confidence always 100%.

---

## Workflow (speedrun)

1. 🕵️ Fetch issue & code  
2. 🧠 Guess which files matter  
3. 🔧 Generate a “solution”  
4. 🧨 Modify your codebase  
5. 🚀 Open PR like nothing happened
